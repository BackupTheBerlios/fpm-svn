CVS

To download the latest CVS development version anonymously, issue the following commands: 

    cvs -d:pserver:anonymous@cvs.fpm.berlios.de:/cvsroot/fpm login
    cvs -z3 -d:pserver:anonymous@cvs.fpm.berlios.de:/cvsroot/fpm co fpm
When prompted for a password, just press enter.

To update an existing check-out, use: 

    cvs -z3 up -dPA
If you want to receive an e-mail every time something has changed in CVS, you can subscribe yourself to the CVS log mailinglist
