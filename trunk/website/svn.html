SVN

To download the latest SVN development version anonymously, issue the following commands: 

    svn co svn://svn.berlios.de/fpm/trunk

To update an existing check-out, use: 

    svn up
If you want to receive an e-mail every time something has changed in SVN, you can subscribe yourself to the old CVS log mailinglist which is now used for Subversion commit messages.
